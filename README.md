# Ride-Hailing Operations & Business Intelligence Dashboard - Iraq

## 📊 Project Overview
An End-to-End Data Analysis project simulating and analyzing operations for a ride-hailing business in Iraq (inspired by Baly). The project covers data generation, database management, SQL analysis, and visual reporting.

## 🛠️ Tools & Technologies Used
- **Python (Faker library):** To generate a realistic dataset of 5,000 customers and 15,000 trips.
- **MySQL Workbench:** Designing a relational schema and conducting data analysis via SQL queries.
- **Power BI:** Creating a star schema data model and designing an interactive management dashboard.

## 📈 Key Insights Addressed
- Tracking Core KPIs: Total Trips (15K), Total Revenue (83M IQD), Net Profit (12.48M IQD), and Average Customer Rating (4.23).
- Geographical Breakdown: Total trips and profit margins distributed across Iraqi provinces (Baghdad, Basra, Karbala, etc.).
- Peak Hours Analysis: Understanding peak operational times using `Trip Hour` to optimize driver allocations.
